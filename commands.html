Basic commands

  • git init: to initialize the git
  • git status: to check the status of the files whether they are tracked or untracked
  • git commit: to commit the message after every step
  • git add filename: to add a particular file into git
  • git add . : to add all the files into the git
  • git log: to see the changes made in the whole process with the help of messages we provided though git commit
  • git reset --hard codecopiedfromlog: this command is used to reset the changes
  • git reset --soft codecopiedfromlog: this is also used to rest the changes with the help of code given in the log command

Commands to push into git hub and to pull into individual desktops

To push into git

  • git remote add origin httpslink of git: which gives the direction to the git
  • git branch -M main: to simplify the complexity the process is divided into branches in which main is an integrated branch
  • git push -u origin main:by using this command we can push the branch with origin into the github
  • git push: if any changes are made there is no need to follow whole procedure again insyead simply use this command to update the changes

    • To pull the files and canges into our desktop

  • git clone linkofthefilefromgithub: create a folder and open command prompt in your desktop and enter the command as mentioned after which the files will be placed into that particular folder you have selected
  • git pull: after cloning the files if admin made any changes in those files there is no need to repeat the whole process intead we use this simple command through which the changes are automatically updated

Branching commands in git

  • git branch: this command is used to check the existing branches
  • git branch test: here test is not a part of command, here we created a branch called test using this command
  • git checkout test: after creating branch through above command there are two branches available which are main and test and by using this command we can access the test branch
  • git merge test: if test branch is working perfectly then switch to main branch using checkout command and use merge command to merge both the main and test branch
